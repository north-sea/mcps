## Entities

### Analytics Import Run (表名: `hermes.analytics_import_runs`)

**描述**: One persisted summary per non-dry-run analytics import. Used by CLI/live smoke/operator diagnostics to understand what was accepted, updated, skipped, unmatched, or failed.

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| import_run_id | UUID | PK | Import run id returned to MCP caller |
| account | TEXT | NOT NULL | WeChat account namespace |
| source | TEXT | NOT NULL | Import source, such as `manual_json`, `manual_csv`, `manual_xls`, `wechat_api`, `browser_automation`, `manual_patch` |
| status | TEXT | NOT NULL, CHECK | `completed`, `completed_with_errors`, `failed` |
| total_rows | INTEGER | NOT NULL DEFAULT 0, CHECK >= 0 | Requested snapshot record count |
| created | INTEGER | NOT NULL DEFAULT 0, CHECK >= 0 | Created snapshot count |
| updated | INTEGER | NOT NULL DEFAULT 0, CHECK >= 0 | Updated snapshot count |
| skipped | INTEGER | NOT NULL DEFAULT 0, CHECK >= 0 | Skipped row count, including P2 rows skipped in MVP |
| unmatched | JSONB | NOT NULL DEFAULT `[]` | Rows that could not resolve to one article |
| errors | JSONB | NOT NULL DEFAULT `[]` | Row-level or import-level structured errors |
| metadata | JSONB | NOT NULL DEFAULT `{}` | Caller import metadata |
| created_at | TIMESTAMPTZ | NOT NULL DEFAULT now() | Creation timestamp |
| updated_at | TIMESTAMPTZ | NOT NULL DEFAULT now() | Last update timestamp |

**索引**:

- `idx_analytics_import_runs_account_created` on `(account, created_at DESC)`
- `idx_analytics_import_runs_source_created` on `(source, created_at DESC)`
- `idx_analytics_import_runs_status_created` on `(status, created_at DESC)`

**状态语义**:

```text
completed              all accepted snapshot rows wrote successfully, no row errors
completed_with_errors  accepted rows wrote successfully, but some rows were skipped/unmatched/invalid
failed                 no accepted writes committed due to validation or DB failure
```

`dry_run` is a response status only and should not persist an `analytics_import_runs` row in MVP.

### Article Metric Snapshot (表名: `hermes.wechat_article_metric_snapshots`)

**描述**: One idempotent overview/conversion metric snapshot for an article at a statistic date/window/source.

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| snapshot_id | UUID | PK | Snapshot row id |
| article_id | UUID | NOT NULL, FK -> `hermes.wechat_articles(article_id)` ON DELETE CASCADE | Stable article binding |
| account | TEXT | NOT NULL | Denormalized account for filtering and diagnostics |
| stat_date | DATE | NOT NULL | Metric statistic date |
| window_label | TEXT | NOT NULL | Window label, e.g. `D+1`, `D+3`, `D+7` |
| source | TEXT | NOT NULL | Import source |
| read_user_count | INTEGER | CHECK >= 0 | Reads/users metric |
| average_stay_seconds | DOUBLE PRECISION | CHECK >= 0 | Average stay duration |
| completion_rate | DOUBLE PRECISION | CHECK 0..1 | Completion ratio |
| new_follow_user_count | INTEGER | CHECK >= 0 | New follows |
| share_user_count | INTEGER | CHECK >= 0 | Share users |
| wow_user_count | INTEGER | CHECK >= 0 | Wow users |
| like_user_count | INTEGER | CHECK >= 0 | Like users |
| favorite_user_count | INTEGER | CHECK >= 0 | Favorites |
| reward_cents | INTEGER | CHECK >= 0 | Reward amount in cents |
| comment_count | INTEGER | CHECK >= 0 | Comments |
| delivered_user_count | INTEGER | CHECK >= 0 | Delivered users |
| account_message_read_user_count | INTEGER | CHECK >= 0 | Reads from account message |
| first_share_user_count | INTEGER | CHECK >= 0 | First share users |
| total_share_user_count | INTEGER | CHECK >= 0 | Total share users |
| share_generated_read_user_count | INTEGER | CHECK >= 0 | Reads generated by sharing |
| missing_fields | TEXT[] | NOT NULL DEFAULT `ARRAY[]::text[]` | Fields absent from source import |
| raw_json | JSONB | NOT NULL DEFAULT `{}` | Raw normalized row/debug payload; omitted from list by default |
| import_run_id | UUID | NULL, FK -> `hermes.analytics_import_runs(import_run_id)` ON DELETE SET NULL | Import provenance |
| collected_at | TIMESTAMPTZ | NULL | Source collection timestamp if known |
| created_at | TIMESTAMPTZ | NOT NULL DEFAULT now() | Creation timestamp |
| updated_at | TIMESTAMPTZ | NOT NULL DEFAULT now() | Last update timestamp |

**唯一约束**:

- `uq_wechat_article_metric_snapshot_identity` on `(article_id, stat_date, window_label, source)`

**索引**:

- `idx_wechat_article_metric_snapshots_account_stat` on `(account, stat_date DESC)`
- `idx_wechat_article_metric_snapshots_article_stat` on `(article_id, stat_date DESC)`
- `idx_wechat_article_metric_snapshots_window_stat` on `(window_label, stat_date DESC)`
- `idx_wechat_article_metric_snapshots_source_stat` on `(source, stat_date DESC)`
- `idx_wechat_article_metric_snapshots_import_run` on `(import_run_id)` where `import_run_id IS NOT NULL`

### Article Channel Daily Metric (表名: `hermes.wechat_article_channel_daily_metrics`)

**描述**: Idempotent per-article per-day per-channel metric rows for traffic source breakdown.

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| metric_id | UUID | PK | Channel metric row id |
| article_id | UUID | NOT NULL, FK -> `hermes.wechat_articles(article_id)` ON DELETE CASCADE | Stable article binding |
| account | TEXT | NOT NULL | Denormalized account for filtering and diagnostics |
| metric_date | DATE | NOT NULL | Channel metric date |
| channel | TEXT | NOT NULL | Source channel, including `全部` if present in source |
| source | TEXT | NOT NULL | Import source |
| read_user_count | INTEGER | CHECK >= 0 | Channel reads/users |
| share_user_count | INTEGER | CHECK >= 0 | Channel share users |
| raw_json | JSONB | NOT NULL DEFAULT `{}` | Raw normalized channel row/debug payload |
| import_run_id | UUID | NULL, FK -> `hermes.analytics_import_runs(import_run_id)` ON DELETE SET NULL | Import provenance |
| created_at | TIMESTAMPTZ | NOT NULL DEFAULT now() | Creation timestamp |
| updated_at | TIMESTAMPTZ | NOT NULL DEFAULT now() | Last update timestamp |

**唯一约束**:

- `uq_wechat_article_channel_daily_identity` on `(article_id, metric_date, channel, source)`

**索引**:

- `idx_wechat_article_channel_daily_account_date` on `(account, metric_date DESC)`
- `idx_wechat_article_channel_daily_article_date` on `(article_id, metric_date DESC)`
- `idx_wechat_article_channel_daily_channel_date` on `(channel, metric_date DESC)`
- `idx_wechat_article_channel_daily_import_run` on `(import_run_id)` where `import_run_id IS NOT NULL`

## Relationships

```text
hermes.wechat_articles 1:N hermes.wechat_article_metric_snapshots
hermes.wechat_articles 1:N hermes.wechat_article_channel_daily_metrics
hermes.analytics_import_runs 1:N hermes.wechat_article_metric_snapshots
hermes.analytics_import_runs 1:N hermes.wechat_article_channel_daily_metrics

Future:
hermes.wechat_articles 1:N hermes.wechat_article_audience_profiles
hermes.analytics_import_runs 1:N hermes.wechat_article_audience_profiles
```

Binding rules:

- `article_id` is the only persisted article binding key in metric tables.
- URL/ref/title matching must not be stored as metric identity.
- `account` is duplicated in analytics rows for query ergonomics and diagnostics, but FK article identity is authoritative.

## DDL Sketch

```sql
CREATE TABLE IF NOT EXISTS hermes.analytics_import_runs (
    import_run_id UUID PRIMARY KEY,
    account TEXT NOT NULL,
    source TEXT NOT NULL,
    status TEXT NOT NULL,
    total_rows INTEGER NOT NULL DEFAULT 0 CHECK (total_rows >= 0),
    created INTEGER NOT NULL DEFAULT 0 CHECK (created >= 0),
    updated INTEGER NOT NULL DEFAULT 0 CHECK (updated >= 0),
    skipped INTEGER NOT NULL DEFAULT 0 CHECK (skipped >= 0),
    unmatched JSONB NOT NULL DEFAULT '[]'::jsonb,
    errors JSONB NOT NULL DEFAULT '[]'::jsonb,
    metadata JSONB NOT NULL DEFAULT '{}'::jsonb,
    created_at TIMESTAMPTZ NOT NULL DEFAULT now(),
    updated_at TIMESTAMPTZ NOT NULL DEFAULT now(),
    CONSTRAINT chk_analytics_import_runs_status
        CHECK (status IN ('completed', 'completed_with_errors', 'failed'))
);

CREATE TABLE IF NOT EXISTS hermes.wechat_article_metric_snapshots (
    snapshot_id UUID PRIMARY KEY,
    article_id UUID NOT NULL REFERENCES hermes.wechat_articles(article_id) ON DELETE CASCADE,
    account TEXT NOT NULL,
    stat_date DATE NOT NULL,
    window_label TEXT NOT NULL,
    source TEXT NOT NULL,
    read_user_count INTEGER CHECK (read_user_count IS NULL OR read_user_count >= 0),
    average_stay_seconds DOUBLE PRECISION CHECK (
        average_stay_seconds IS NULL OR average_stay_seconds >= 0
    ),
    completion_rate DOUBLE PRECISION CHECK (
        completion_rate IS NULL OR (completion_rate >= 0 AND completion_rate <= 1)
    ),
    new_follow_user_count INTEGER CHECK (
        new_follow_user_count IS NULL OR new_follow_user_count >= 0
    ),
    share_user_count INTEGER CHECK (share_user_count IS NULL OR share_user_count >= 0),
    wow_user_count INTEGER CHECK (wow_user_count IS NULL OR wow_user_count >= 0),
    like_user_count INTEGER CHECK (like_user_count IS NULL OR like_user_count >= 0),
    favorite_user_count INTEGER CHECK (
        favorite_user_count IS NULL OR favorite_user_count >= 0
    ),
    reward_cents INTEGER CHECK (reward_cents IS NULL OR reward_cents >= 0),
    comment_count INTEGER CHECK (comment_count IS NULL OR comment_count >= 0),
    delivered_user_count INTEGER CHECK (
        delivered_user_count IS NULL OR delivered_user_count >= 0
    ),
    account_message_read_user_count INTEGER CHECK (
        account_message_read_user_count IS NULL OR account_message_read_user_count >= 0
    ),
    first_share_user_count INTEGER CHECK (
        first_share_user_count IS NULL OR first_share_user_count >= 0
    ),
    total_share_user_count INTEGER CHECK (
        total_share_user_count IS NULL OR total_share_user_count >= 0
    ),
    share_generated_read_user_count INTEGER CHECK (
        share_generated_read_user_count IS NULL OR share_generated_read_user_count >= 0
    ),
    missing_fields TEXT[] NOT NULL DEFAULT ARRAY[]::text[],
    raw_json JSONB NOT NULL DEFAULT '{}'::jsonb,
    import_run_id UUID REFERENCES hermes.analytics_import_runs(import_run_id) ON DELETE SET NULL,
    collected_at TIMESTAMPTZ,
    created_at TIMESTAMPTZ NOT NULL DEFAULT now(),
    updated_at TIMESTAMPTZ NOT NULL DEFAULT now(),
    CONSTRAINT uq_wechat_article_metric_snapshot_identity
        UNIQUE (article_id, stat_date, window_label, source)
);

CREATE TABLE IF NOT EXISTS hermes.wechat_article_channel_daily_metrics (
    metric_id UUID PRIMARY KEY,
    article_id UUID NOT NULL REFERENCES hermes.wechat_articles(article_id) ON DELETE CASCADE,
    account TEXT NOT NULL,
    metric_date DATE NOT NULL,
    channel TEXT NOT NULL,
    source TEXT NOT NULL,
    read_user_count INTEGER CHECK (read_user_count IS NULL OR read_user_count >= 0),
    share_user_count INTEGER CHECK (share_user_count IS NULL OR share_user_count >= 0),
    raw_json JSONB NOT NULL DEFAULT '{}'::jsonb,
    import_run_id UUID REFERENCES hermes.analytics_import_runs(import_run_id) ON DELETE SET NULL,
    created_at TIMESTAMPTZ NOT NULL DEFAULT now(),
    updated_at TIMESTAMPTZ NOT NULL DEFAULT now(),
    CONSTRAINT uq_wechat_article_channel_daily_identity
        UNIQUE (article_id, metric_date, channel, source)
);
```
